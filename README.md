## 🛠 Set up 🛠

### Yarn
```
yarn init
```

### Vite
```
yarn add vite
```
package.json
```json
"scripts": {
  "dev": "vite"
}
```

### Eslint
```
npm init @eslint/config
```

### Jest
```
yarn add --dev jest
```
package.json
```json
"scripts": {
    "test": "jest"
}
```

### Husky
```
yarn add husky-init --yarn2 && yarn
```
package.json
```json
"scripts": {
  "prepare": "husky install"
}
```

### Prettier
```
yarn add --dev --exact prettier
node --eval "fs.writeFileSync('.prettierrc','{}\n')"
```
package.json
```json
"scripts": {
  "prettier": "prettier . --write"
}
```

### Eslint-config-prettier
```
npm install --save-dev eslint-config-prettier
```
.eslintrc.json
```json
"extends": [
    "eslint:recommended",
    "prettier"
]
```

### Sass
```
npm install -g sass
```
package.json
```json
"scripts": {
  "sass": "sass --watch style/style.scss:style/style.css"
}
```
